HTML

Important/Frequently Used Tags

Tags Decsription
<html>, <head>, <title>, <body>, <h1>-<h6>, <div>, <ul>, <ol>,<li>, <em>, <strong> Tags used Frequently
<header>, <nav>, <main>, <footer>, <body>, <audio>, <video>, <embed>, <section>, <article>, <figure>, <figcaption>, <aside>, <br>, <span> Semantic Tags
<video>,<figure>, <image> Tags used for videos and images, embed can be used for videos and images.

Image tags are self closing, videos need closing tag.

<video src="link" controls> video not supported </video> (instead of controls you can use autoplay or loop)

<img src="link" alt=text />

<figure <img src="link"> <figcaption> text </figcaption> </figure>
<table>, <thead>, <th>, <tbody>, <tr>, <td>, <tfoot> Main tags for table content
Table Attributes: colspan & rowspan additional cell space can be used for columns and rows

<td colspan='2'> column 1 (will span 2 columns) </td>

<td rowspan='3'> row 1 (will span 3 rows) </td>
Attributes: anchor, href, src, class, id Attributes have name=value

<a href="link" target="_blank"> </a> (External webpage, Absolute Path)

<a href="./home.html"> Home </a> (Internal webpage, Relative path, info stored in other files)

<ul>
<li> <a href="#home"> Home </a> </li>
<li> <a href="#about"> About </a> </li>
<li> <a href="#contact"> Contact </a> </li>
</ul>
<!--comment-->, <div> comment and div tags are good for keeping html and css code clean and in Order
